## Getting Started
1. **Fork & Clone**
   ```bash
   git clone https://github.com/siswoyo-dev/redis-prayer-times.git
   cd redis-prayer-times
   ```
2. **Install dependencies**
   ```bash
   pnpm install
   ```
3. **Set up Redis** locally or via a hosted service.  
4. Create a `.env`:
   ```env
   REDIS_HOST=
   REDIS_PORT=
   REDIS_USERNAME=
   REDIS_PASWORD=
   JWT_SECRET=
   ```
5. **Run tests**
   ```bash
   pnpm dev
   ```
